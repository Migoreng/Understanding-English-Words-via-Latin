**fugitive**
* 語源: ラテン語 "fugitivus" (逃げる) ← "fugere" (逃げる)
* 日本語訳: 逃亡者、逃げ去る、つかの間の

**fumigate**
* 語源: ラテン語 "fumigatus" (燻蒸された) ← "fumigare" (煙でいぶす)
* 日本語訳: 燻蒸する、消毒する、燻製にする

**fund**
* 語源: ラテン語 "fundus" (底、基金) ← ラテン語 "fundus" (土地)
* 日本語訳: 基金、資金、資金を供給する

**fundamental**
* 語源: ラテン語 "fundamentalis" (基礎の) ← "fundare" (基礎を置く) + 英語接尾辞 "-al" (属性)
* 日本語訳: 基本的な、根本的な、本質的な

**fund-raising**
* 語源: ラテン語 "fund" (基金) ← "fundus" (底、基礎) + 英語 "raising" (募集する) ← ゲルマン語 "rāisan" (上げる)
* 日本語訳: 資金調達、募金活動、資金募集

**fungus**
* 語源: ラテン語 "fungus" (キノコ) ← ラテン語 "fungus" (菌類)
* 日本語訳: 菌類、キノコ、真菌

**furtive**
* 語源: ラテン語 "furtivus" (盗みの) ← "furtum" (盗み)
* 日本語訳: こっそりした、隠れた、盗むような

**fury**
* 語源: ラテン語 "furia" (怒り) ← "furere" (怒る)
* 日本語訳: 激怒、怒り、猛烈さ

**fusion**
* 語源: ラテン語 "fusio" (溶解) ← "fundere" (溶かす)
* 日本語訳: 融合、核融合、混合
