**quadrangle**
* 語源: ラテン語 "quadrangulus" (四角形の) ← "quadri" (四) + "angulus" (角)
* 日本語訳: 四角形、四辺形、四角い中庭

**quaint**
* 語源: ラテン語 "quintus" (第五の) ← フランス語 "cointe" (優雅な)
* 日本語訳: 古風な、珍しい、趣のある

**qualification**
* 語源: ラテン語 "qualificatio" (適性の判定) ← "qualificare" (適格にする)
* 日本語訳: 資格、適格性、限定条件

**qualified**
* 語源: ラテン語 "qualificatus" (適格になった) ← "qualificare" (適格にする)
* 日本語訳: 資格のある、適格の、限定的な

**quantum**
* 語源: ラテン語 "quantum" (量) ← "quantus" (どれだけの)
* 日本語訳: 量子、量、分量

**quarrel**
* 語源: ラテン語 "querrella" (苦情) ← "querere" (文句を言う)
* 日本語訳: けんか、争い、異議を唱える

**quash**
* 語源: ラテン語 "quassatus" (砕かれた) ← "quassare" (砕く)
* 日本語訳: 破棄する、無効にする、鎮圧する

**querulous**
* 語源: ラテン語 "querulus" (文句が多い) ← "querere" (文句を言う) + 英語接尾辞 "-ous" (である)
* 日本語訳: 文句が多い、愚痴っぽい、不平な

**query**
* 語源: ラテン語 "quaerere" (質問する) ← ラテン語 "quaerere" (質問する)
* 日本語訳: 質問、疑問、問い合わせる

**quest**
* 語源: ラテン語 "quaestio" (探索) ← "quaerere" (探す)
* 日本語訳: 探索、追求、冒険

**questionnaire**
* 語源: ラテン語 "quaestio" (質問) + フランス語接尾辞 "-naire" (～に関する人)
* 日本語訳: 質問票、アンケート、調査票

**quibble**
* 語源: ラテン語 "quibble" (言い言葉) ← フランス語 "quibble" (些細なことで言い争う)
* 日本語訳: つまらないことで言い争う、屁理屈をこねる

**quintessence**
* 語源: ラテン語 "quintaessentia" (第五の本質) ← "quintus" (第五の) + "essentia" (本質)
* 日本語訳: 最精粋、本質、極致

**quorum**
* 語源: ラテン語 "quorum" (その者たちの) ← "quorum" (～のうちの)
* 日本語訳: 定足数、法定数、所要人数

**quota**
* 語源: ラテン語 "quota" (どの程度の) ← "quotus" (何番目の)
* 日本語訳: 割当量、定員、配分量
