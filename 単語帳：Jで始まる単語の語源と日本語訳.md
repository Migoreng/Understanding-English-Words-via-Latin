**janitor**
* 語源: ラテン語 "janitor" (戸番) ← "janua" (戸、入口)
* 日本語訳: 門番、管理人、用務員

**jejune**
* 語源: ラテン語 "jejunus" (空の、貧困な)
* 日本語訳: つまらない、退屈な、栄養のない

**jeopardy**
* 語源: ラテン語 "jeopardium" (危険) ← フランス語 "jeu parti" (同等の賭け)
* 日本語訳: 危険、危機、危地

**jest**
* 語源: ラテン語 "gesta" (行為) ← "gestum" (行為)
* 日本語訳: 冗談、ふざけ、戯言

**jestingly**
* 語源: "jest" + 英語接尾辞 "-ingly" (副詞化)
* 日本語訳: 冗談めかして、ふざけて、戯れに

**jettison**
* 語源: ラテン語 "jectio" (投げること) ← "jacere" (投げる)
* 日本語訳: 投棄する、投げ捨てる、荷を落とす

**jocular**
* 語源: ラテン語 "jocularis" (冗談の) ← "joculus" (小さな冗談) + 英語接尾辞 "-ar" (属性)
* 日本語訳: ユーモアのある、冗談好きな、ふざけた

**joint**
* 語源: ラテン語 "junctus" (結合された) ← "jungere" (結合する)
* 日本語訳: 関節、継ぎ目、連合の

**jostle**
* 語源: ラテン語 "jostulare" (衝突させる) ← "jouste" (突き)
* 日本語訳: ぶつかる、押しのける、肘鉄を食わせる

**journalism**
* 語源: フランス語 "journal" (日記) + 英語接尾辞 "-ism" (主義)
* 日本語訳: ジャーナリズム、新聞業、報道

**joust**
* 語源: ラテン語 "justare" (突く) ← フランス語 "jouste" (槍試合)
* 日本語訳: 槍試合、騎士の競技、決闘

**jovial**
* 語源: ラテン語 "jovialis" (木星の) ← "Iuppiter" (木星、ジュピター)
* 日本語訳: 陽気な、明朗な、楽しい

**joyous**
* 語源: ラテン語 "gaudiosus" (喜びに満ちた) ← "gaudium" (喜び)
* 日本語訳: 喜びに満ちた、幸せな、嬉しい

**jubilant**
* 語源: ラテン語 "jubilans" (歓呼している) ← "jubilare" (歓呼する)
* 日本語訳: 歓喜に満ちた、大喜びの、勝利に沸く

**jubilation**
* 語源: ラテン語 "jubilatio" (歓呼) ← "jubilare" (歓呼する)
* 日本語訳: 歓喜、歓呼、大喜び

**judicial**
* 語源: ラテン語 "judicialis" (裁判の) ← "judex" (裁判官) + 英語接尾辞 "-ial" (属性)
* 日本語訳: 司法の、裁判の、判断的な

**judicious**
* 語源: ラテン語 "judiciosus" (判断力のある) ← "judicium" (判断)
* 日本語訳: 判断力のある、分別のある、思慮深い

**juggle**
* 語源: ラテン語 "joculari" (冗談を言う) ← "joculus" (冗談)
* 日本語訳: 玉を回す、ジャグリングする、巧みに操る

**juicy**
* 語源: ラテン語 "jus" (汁) + 英語接尾辞 "-y" (である)
* 日本語訳: 果汁の多い、ジューシーな、興味深い

**junction**
* 語源: ラテン語 "junctio" (結合) ← "jungere" (結合する)
* 日本語訳: 接合、交差点、結合点

**juncture**
* 語源: ラテン語 "junctura" (結合) ← "jungere" (結合する)
* 日本語訳: 接合、時期、局面

**jurisdiction**
* 語源: ラテン語 "jurisdictio" (司法管轄権) ← "jus" (法) + "dicere" (言う)
* 日本語訳: 管轄権、司法管轄権、裁判権

**juror**
* 語源: ラテン語 "jurator" (宣誓者) ← "jurare" (誓う)
* 日本語訳: 陪審員、陪審団の構成員

**jury**
* 語源: ラテン語 "jurata" (宣誓者) ← "jurare" (誓う)
* 日本語訳: 陪審団、陪審員、判定委員会

**juvenile**
* 語源: ラテン語 "juvenilis" (若い) ← "juvenis" (若者) + 英語接尾辞 "-ile" (である)
* 日本語訳: 少年の、幼稚な、青少年

**juxtapose**
* 語源: ラテン語 "juxta" (そばに) + "ponere" (置く)
* 日本語訳: 並置する、隣同士に置く、対比させる
