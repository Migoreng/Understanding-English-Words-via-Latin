**habitable**
* 語源: ラテン語 "habitabilis" (住める) ← "habitare" (住む) + 英語接尾辞 "-able" (できる)
* 日本語訳: 住める、居住可能な、適した

**habitat**
* 語源: ラテン語 "habitat" (彼は住む) ← "habitare" (住む)
* 日本語訳: 生息地、住処、自然環境

**haughty**
* 語源: ラテン語 "altus" (高い) ← フランス語 "hault" (高い)
* 日本語訳: 傲慢な、高慢な、尊大な

**heir**
* 語源: ラテン語 "heres" (相続人) ← ラテン語 "haerere" (保持する)
* 日本語訳: 相続人、跡継ぎ、後継者

**herbicide**
* 語源: ラテン語 "herba" (草) + "caedere" (切る)
* 日本語訳: 除草剤、雑草除去剤

**herbivore**
* 語源: ラテン語 "herba" (草) + "vorare" (食べる)
* 日本語訳: 草食動物、草食性動物

**hereditary**
* 語源: ラテン語 "hereditarius" (相続の) ← "hereditas" (相続) + 英語接尾辞 "-ary" (属性)
* 日本語訳: 遺伝の、世襲の、相続による

**heredity**
* 語源: ラテン語 "hereditas" (相続) ← "heres" (相続人)
* 日本語訳: 遺伝、相続、家系

**heritage**
* 語源: ラテン語 "heres" (相続人) ← フランス語 "heritage" (相続)
* 日本語訳: 遺産、文化遺産、伝統

**hesitant**
* 語源: ラテン語 "hesitans" (ためらう) ← "haesitare" (ためらう)
* 日本語訳: ためらう、躊躇する、不確かな

**hesitate**
* 語源: ラテン語 "haesitatus" (ためらわれた) ← "haesitare" (ためらう)
* 日本語訳: ためらう、躊躇する、戸惑う

**hiatus**
* 語源: ラテン語 "hiatus" (開くこと) ← "hiare" (開く)
* 日本語訳: 間隙、裂け目、中断

**hibernate**
* 語源: ラテン語 "hibernatus" (冬眠した) ← "hibernare" (冬を過ごす)
* 日本語訳: 冬眠する、冬を過ごす、眠る

**histrionics**
* 語源: ラテン語 "histrionicus" (役者の) ← "histrio" (役者)
* 日本語訳: 大げさな演技、わざとらしい行動、舞台的

**homage**
* 語源: ラテン語 "hominium" (人間) ← "homo" (人間)
* 日本語訳: 敬意、崇拝、忠誠の誓い

**homicide**
* 語源: ラテン語 "homo" (人間) + "caedere" (切る、殺す)
* 日本語訳: 殺人、殺人犯、人殺し

**honesty**
* 語源: ラテン語 "honestas" (名誉) ← "honestus" (名誉ある)
* 日本語訳: 正直さ、誠実さ、廉潔さ

**horrible**
* 語源: ラテン語 "horribilis" (恐ろしい) ← "horrere" (身震いする) + 英語接尾辞 "-ible" (できる)
* 日本語訳: 恐ろしい、ひどい、嫌悪すべき

**horrify**
* 語源: ラテン語 "horrificus" (恐ろしい) ← "horrere" (身震いする) + "facere" (作る)
* 日本語訳: ぞっとさせる、恐怖させる、嫌悪させる

**hortatory**
* 語源: ラテン語 "hortatorius" (勧めるような) ← "hortari" (勧める)
* 日本語訳: 勧告的な、勧奨的な、激励的な

**horticulture**
* 語源: ラテン語 "hortus" (庭園) + "cultura" (耕作)
* 日本語訳: 園芸、園芸学、庭園栽培

**hostage**
* 語源: ラテン語 "hosticum" (敵) ← "hostis" (敵)
* 日本語訳: 人質、担保、拘束される人

**hostile**
* 語源: ラテン語 "hostilis" (敵の) ← "hostis" (敵) + 英語接尾辞 "-ile" (である)
* 日本語訳: 敵対的な、敵意のある、非友好的な

**humane**
* 語源: ラテン語 "humanus" (人間らしい) ← "homo" (人間)
* 日本語訳: 思いやりのある、人道的な、慈悲深い

**humanitarian**
* 語源: ラテン語 "humanus" (人間らしい) + 英語接尾辞 "-itarian" (主義者)
* 日本語訳: 人道主義者、人道的な、慈善家

**humanity**
* 語源: ラテン語 "humanitas" (人間性) ← "humanus" (人間らしい)
* 日本語訳: 人類、人間性、人道心

**humanlike**
* 語源: ラテン語 "humanus" (人間らしい) + 英語 "like" (～のような)
* 日本語訳: 人間のような、人間に似た

**humble**
* 語源: ラテン語 "humilis" (低い) ← "humus" (地面)
* 日本語訳: 謙虚な、低い、卑しい

**humiliate**
* 語源: ラテン語 "humiliatus" (低くされた) ← "humiliare" (低くする)
* 日本語訳: 屈辱を与える、面目を潰す、辱める

**hydrocarbon**
* 語源: ギリシャ語 "hydro" (水) + 英語 "carbon" (炭素)
* 日本語訳: 炭化水素、水素化炭素
