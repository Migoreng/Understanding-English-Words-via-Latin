* 日本語訳: 犯人引渡しする、身柄を引き渡す

**extradition**
* 語源: "extradite" + 英語接尾辞 "-tion" (行為、状態)
* 日本語訳: 犯人引渡し、身柄引き渡し

**extraneous**
* 語源: ラテン語 "extraneous" (外側の) ← "extrinsecus" (外側の)
* 日本語訳: 無関係な、関係のない、外部からの

**extrapolate**
* 語源: ラテン語 "extra" (外に) + "polatus" (平にされた)
* 日本語訳: 外挿する、推測する、推定する

**extraterrestrial**
* 語源: ラテン語 "extra" (外に) + "terrestrial" (地上の)
* 日本語訳: 地球外の、宇宙人、異星人

**extravagant**
* 語源: ラテン語 "extravagans" (範囲を超えた) ← "extra" (外に) + "vagans" (さまよう)
* 日本語訳: 浪費的な、贅沢な、法外な

**extreme**
* 語源: ラテン語 "extremus" (最後の、最も遠い) ← "exterus" (外側の)
* 日本語訳: 極端な、最大限の、過激な

**extremism**
* 語源: "extreme" + 英語接尾辞 "-ism" (主義)
* 日本語訳: 過激主義、極端主義、激進主義

**extremity**
* 語源: ラテン語 "extremitas" (端) ← "extremus" (最後の)
* 日本語訳: 極限、手足、最後の手段

**extricable**
* 語源: ラテン語 "extricabilis" (助け出せる) ← "extricare" (助け出す) + 英語接尾辞 "-able" (できる)
* 日本語訳: 救い出せる、脱出可能な、助け出せる

**extricate**
* 語源: ラテン語 "extricatus" (助け出された) ← "extricare" (助け出す)
* 日本語訳: 救い出す、脱出させる、解放する

**extrinsic**
* 語源: ラテン語 "extrinsecus" (外側の) ← "extra" (外に) + "secus" (そば)
* 日本語訳: 外的な、外部からの、本質的でない

**extrovert**
* 語源: ラテン語 "extra" (外に) + "vertere" (向ける)
* 日本語訳: 外向型の人、社交的な人

**exuberant**
* 語源: ラテン語 "exuberans" (豊かに流れ出る) ← "exuberare" (豊かに流れ出る)
* 日本語訳: 生き生きした、活気のある、有り余る

**exude**
* 語源: ラテン語 "exudare" (滲み出る) ← "ex" (外に) + "sudare" (汗をかく)
* 日本語訳: 滲み出す、放出する、漂わせる

**exult**
* 語源: ラテン語 "exsultare" (跳び上がる) ← "ex" (外に) + "saltare" (跳ぶ)
* 日本語訳: 歓喜する、喜び勇む、得意になる

**exultant**
* 語源: ラテン語 "exsultans" (跳び上がっている) ← "exsultare" (跳び上がる)
* 日本語訳: 歓喜した、喜び勇んだ、勝利の
